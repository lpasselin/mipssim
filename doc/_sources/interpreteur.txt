Module d'interprétation
=======================

Utilisation du module
---------------------

Le module d'interprétation s'instancie avec le chemin du code source assembleur MIPS 64 bits en paramètre comme suit::

    from src.interpreter import *
    interpreter = mips_interpreteur(source_file)

L'interpréteur supporte l'étiquettage (*label*) d'une ligne.

La syntaxe supportée est la suivante::

    [label:] Mnémonique  param1,param2[,param3]

Les éléments entre crochets sont optionnels.

Les paramètres supportés sont les suivants:

- Registre entier (R0* à R31)
- Registre flottant (F0 à R31)
- Adresse mémoire (noté: offset(registre contenant l'adresse), ie: 0(R1)  )
- Étiquette (*label*) dans le cas des branch

\* Attention au registre R0, c'est un registre à lecture seule.

Mnémoniques assembleur supportés
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Les mnémoniques assembleur MIPS 64 bits supportés par l'interpréteurs sont les suivants:

- LD
- L.D
- SD
- S.D
- ADD.D
- SUB.D
- MUL.D
- DIV.D
- DADD
- DADDU
- DADDI
- DADDIU
- DSUB
- DSUBU
- DMUL
- DMULU
- DDIV
- DDIVU
- AND
- BEQZ
- BNEZ
- BEQ
- BNE
- J
